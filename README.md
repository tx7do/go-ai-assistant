# go-ai-assistant

This is a [OpenAI Assistant](https://platform.openai.com/docs/assistants/overview) application.

## Install Wails CLI

```bash
go install github.com/wailsapp/wails/v2/cmd/wails@latest
```

## Run

```bash
wails dev
```

## Build

```bash
wails build
```

Keep Windows Console:

```bash
wails build -windowsconsole -debug
```
